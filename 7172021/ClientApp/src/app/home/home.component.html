Welcome to this game!

This is a simple app using .NET and Angular to practice for a job interview.
The end goal is a turn based strategy game (a la Risk, but with some added elements, or possible CK2) that will support user acocunts, custom map making, and have default maps as well. The actual game will be world domination based, but it may grow in scope.
Data will be tracked with SQL, C# will handle requests and some interpreting, Angular will present views. Bootstrap will be used for some parts. Also some research into Node will be considered (sockets?)
